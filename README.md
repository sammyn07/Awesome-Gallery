# Awesome Gallery
 A simple, mobile-friendly image gallery with search, fullscreen, and download features.
 ## What It Does
 - Loads beautiful images dynamically from Unsplash API
 - Search for any image topic
 - View images in fullscreen
 - Download images with one tap
 - "Show More" button for extra content without searching
 ## How to Use
 1. Visit the website: [https://sammyn07.github.io/Awesome-Gallery/](https://sammyn07.github.io/Awesome-Gallery/)
 2. Use the search bar to find specific images
 3. Tap an image to open fullscreen (tap overlay or "Ã—" to close)
 4. Tap "Download" to save images to your device
 5. Tap "Show More" to load additional images
